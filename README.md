# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 导入调试视频
- 开发工具安装包
- 参考导入调试视频
- 讲解视频

## 适用场景
- 演出购票平台搭建
- 在线票务销售系统
- 文化活动组织管理

## 技术特点
- B/S架构，MVC设计模式
- 采用Java、SpringBoot、Vue技术栈
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
